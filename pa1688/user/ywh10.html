YWH10 IP Phone
  
There are many kind of PA1688 IP phones made by Yuxin, all use their own software slightly different from our standard reference design. However, the operation of those phones are almost the same with the standard design.
Standard reference design IP phone web manual: SIP IAX2 H323 MGCP. Menu manual: SIP IAX2 H323 MGCP. Telnet manual: SIP IAX2 H323 MGCP.
YWH10 uses its own software with name like ywh10_sip_us_168000.bin. Both Yuxin and Palmmicro can provide the software. Although it looks like Primeworx P100 and IPH_E00_2G, the only common thing about those phones is that they all use perhaps the cheapest PSTN phone case available in the market. Large
