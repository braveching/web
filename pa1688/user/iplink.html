IPLINK IP Phone
 
Iplink made this modified PA168T IP phone. The difference is the same as 5111PHONE vs PA168S. It uses iplink software, the typical compile command line is mk iplink sip us.
The phone uses menu (SIP IAX2 H323 MGCP) slightly different from our standard design, but the usage of web (SIP IAX2 H323 MGCP) and telnet SIP IAX2 H323 MGCP) are the same.
