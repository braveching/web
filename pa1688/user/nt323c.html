NT323C 1-port FXS Gateway
  
ForeRun NT323C 1-port FXS gateway is a modified version of PA168Q hardware design. Although ForeRun continued to use standard PA168Q software name for their upgrade file, as they did not share what they had changed in hardware, we can not provide upgrade files for this device.
