DP3000 1-port FXS Gateway
  
DareGlobal DP3000 is the same as standard PA168P design, and can use the software directly.
Standard 1-port FXS Gateway reference design web manual: SIP IAX2 H323 MGCP. Telnet manual: SIP IAX2 H323 MGCP.
From above 3 PCB images we can see this DP3000 used Realtek ALC202A as its AC97 chip. It is rarely found in PA1688 devices. Although we always say it is not necessary to update page0 software, but if you found this ATA has voice problem, we sugguest to update page0 safe mode software.
Safe mode page0 software is not available for download, please write email to us.
