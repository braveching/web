Camera Manager(CamMan) User Manual - Quick Start
  
Overview
CamMan(Palmmicro Camera Manager) is a PA6488 support tool running on Windows system. It servers the following purposes:
1. It is a network camera viewer software, PA6488 IP camera owners can use it with their devices. As PA6488 camera is based on open standards like SIP and H.264, we hope and believe there will be more third party viewers on those platforms.
2. It is a PA6488 IP camera simulation software, Windows users can enjoy the network camera function with their USB camera on PC. When PC camera user feels it is time to get a standalone and high performance device, then it is time to consider a PA6488 solution based IP camera.
3. It can be used as a raw image/video data viewer.
  
Download
Compiled CamMan.exe is located in PA6488 software API pa6488\bin directory. You can also download a seperated copy here. The current version number is 1.0. Our version number on PC software always goes a lot ahead of those embeded related.
This software does not have auto update function, so please come back often to see what is new.
  
Install
Actually there is no special install process. As this program is compiled with Microsoft Visual Studio 2008 (VC9) and Windows SDK v7.1, you might need to download and install related VC9 redistributable package from Microsoft first.
