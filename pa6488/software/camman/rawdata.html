Camera Manager(CamMan) User Manual - Raw Data Viewer
  
Start
In left panel, under "Local Cameras", select "Camera Simulater", click right mouse for menu, and select "View Camera". A dialog will prompt you to select file and set the format details. After that, it will display the file repeatly.
  
Format
Currently support YV12, IYUV(I420), Y800(Y8/GREY), YUY2, UYVY.
