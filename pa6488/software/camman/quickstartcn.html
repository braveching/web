CamMan摄像头管理器使用手册 - 快速上手
  
概述
Palmmicro CamMan摄像头管理器在Windows平台上运行，是一个PA6488辅助支持工具。它有以下用途：
1. 它是一个网络摄像头的监控软件，配合PA6488方案的IP摄像头工作。由于PA6488方案的摄像头使用像SIP和H.264这样的标准协议，我们希望并且相信会有更多第三方的监控软件在各种平台上配合使用。
2. 它是一个PA6488方案IP摄像头的仿真软件。Windows用户可以通过它直接在PC上用USB摄像头实现网络摄像头功能。当用户觉得希望有一个不需要开关PC机而且高性能的设备的时候，就可以考虑基于PA6488方案的网络摄像头了。
3. 它可以用来查看原始图像数据。.
  
下载
编译好的CamMan.exe放在PA6488软件API的pa6488\bin子目录下，也可以在这里单独下载。目前版本号为1.0，我们PC软件的版本号总是远远超前于对应的嵌入式软件版本号。
这个软件没有自动升级功能，请经常回到这里察看更新内容。
  
安装
事实上并没有单独的安装过程。但由于它是用Microsoft Visual Studio 2008 (VC9)和Windows SDK v7.1编译的，用户可能需要从微软下载和安装VC9的可再发行组件包。
